# Gin + React TypeScript Fullstack Starter
# Tien-Wei Hsu 用來練習的 歡迎自取

這是一個以 **Gin（Go）後端** 搭配 **React TypeScript 前端** 所構建的全端練習專案。  
專案目標是打造一個簡潔、模組化且可擴充的開發架構，並示範以下技術的整合方式：

- ⚡ **Gin** — 高效、輕量級 Go Web Framework
- 🗄️ **GORM** — ORM，用於模型管理與資料庫操作
- 🔐 **JWT Authentication** — 使用 JSON Web Token 來保護 API
- 🎨 **React + TypeScript** — 前端使用 Vite 建立，擁有型別安全與快速開發體驗
- 💅 **Tailwind CSS +（可加 Shadcn UI）** — 現代化 UI 開發方式
- 🔗 **前後端 API 串接** — 使用 REST API 完成 CRUD 與登入流程

此專案適合作為：
- 想學習 Gin 與 React TS 全端開發的入門模板  
- 想打造公司內部管理系統的基礎架構  
- 想展示在 GitHub 上的作品集專案  

---

## 🔥 功能特色

- 使用 GORM 建立模型與資料庫（SQLite / MySQL 可切換）
- JWT 登入驗證（Protect Routes / Token 驗證）
- 清晰的後端分層架構：`controllers / routes / middleware / models / config`
- React 前端具備登入、登出、資料取得等基本功能
- 支援 CORS，可直接與前端本地伺服器（5173）串接
